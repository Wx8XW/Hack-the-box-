#Hack The Box 

This repository contains my notes and write-ups for machines and labs from Hack The Box.
The goal is to document my learning process while practicing penetration testing and cybersecurity skills.

Note: These write-ups are for educational purposes only.

##Skills Practiced
During these labs I practice:
Network enumeration
Web application testing
Exploitation techniques
Privilege escalation
Linux and system enumeration
Post-exploitation

##Tools Use
Some of the tools commonly used in these labs include:
Nmap – network and port scanning
Gobuster – directory brute forcing
Netcat – reverse shells and connections
Burp Suite – web application testing
LinPEAS – privilege escalation enumeration
Wireshark – network analysis

Through these labs I aim to improve:
Offensive security skills
Problem solving in CTF environments
Linux system knowledge
Understanding of real-world vulnerabilities
